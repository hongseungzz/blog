## 리스트 컴프리헨션(list comprehension)
- 반복문을 한 줄에 처리할 수 있음


```python
fruits = ['apple', 'kiwi', 'banana']
newlist = []

# 알파벳 a가 있는 과일만 추출 후, 새로운 리스트에 담기
for fruit in fruits :
  # print(fruit)
  if "a" in fruit : 
    newlist.append(fruit)
print(newlist)
```

    ['apple', 'banana']
    


```python
newlist = [fruit for fruit in fruits if "a" in fruit]
print(newlist)
```

    ['apple', 'banana']
